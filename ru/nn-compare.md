# Детерминированность фрэймворков

1. Раздел содержит информацию о потенциальном наличии детерминированного 
режима у LLM различных производителей  на основе используемых фрйэмворков.
2. Раздел в разработкие.



# Перечень фрэмворков
| Фреймворк | Детерминированный режим  |
|:--        |:--                       |
| PyTorch   | Нет                      |
| vLLM      | Нет                      |
| SGLang    | Да                       |
