# Archcode - архитектура как код











Ресурс содержит документацию как код на основе[EPL](/entities/epl/default.md ""). 
[Редактор archcode](/entities/archcode-editor/default.md "") пишут исходники документов в различных 
[форматах](/entities/archcode-format/default.md ""), включая в них ссылки на [Сущности](/entities/entity/gram-number=plural.md "").

Исходники документации при сборке конвертируются в готовый файловый проект, 
загружаемый в репозитории [git](/entities/git/default.md ""), [wiki](/entities/wiki/default.md ""), либо web ресурс.
