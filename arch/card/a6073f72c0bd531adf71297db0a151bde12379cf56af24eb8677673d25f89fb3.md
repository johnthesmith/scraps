# Сущность (entity)



---

Базовая самоописывающаяся идентифицируемая сущность.
Порождает все иные понятия и сущности через их типизацию.


---

1. Каждая сущность должна быть описана;
2. Описание не должно использовать собственное имя сущности;
3. Описание не должно ссылаться на сущности нижних уровней;
4. Сущности объединяются в единую онтологию через отношения типизации;
5. При равных гипотезах выбирается простейшая (бритва Оккама);
6. Структура онтологии должна быть минимально необходимой.
