# Корневая сущность

1. Содержит единственную сущность `entity`, которая является корнем онтологии EPL.



## Файл

- `0/entity.yaml` — декларация и описание сущности `entity`.



## Описание

`entity` — базовая самоописывающаяся идентифицируемая сущность. Порождает все иные понятия онтологии.



## Правила уровня 0

1. Уровень 0 содержит только сущность `entity`.
2. Все остальные сущности прямо или косвенно являются подтипами `entity`.
3. Описание `entity` не ссылается на другие сущности (вышестоящих нет).
